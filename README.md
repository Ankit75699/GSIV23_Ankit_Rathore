
## Run Locally

Clone the project

```bash
  git clone https://github.com/Ankit75699/GSIV23_Ankit_Rathore.git
```

Go to the project directory

```bash
  cd movie-browser
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

To Run Test

```bash
  npm run test
```
